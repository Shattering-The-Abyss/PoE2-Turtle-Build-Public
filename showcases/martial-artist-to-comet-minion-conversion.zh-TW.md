## 真實性邊界

這是一份「能不能轉、怎麼轉、值不值得轉」的轉換評估，不是完整成品 BD。

目前角色是 81 級 Monk / Martial Artist。目標路線來自影片截圖，看起來是偏 Witch / Blood Mage 的召死彗星體系。下面會把目前角色硬資料和轉換推論分開說。

## 一句話診斷

這不是小洗點。目前角色是 `Killing Palm / Quarterstaff / Attack / Crit` 武聖殼子；目標是 `Minion / Trigger / Spell / Spirit / Curse` 殼子。能轉，但更像換引擎，不是調參數。

## 武僧哪個昇華適合？

如果一定要留在 Monk：

```text
Invoker > Acolyte of Chayula > Martial Artist
```

### Invoker

Invoker 是最不彆扭的 Monk 選擇。它和元素、法術、資源、防禦的關係更接近召死彗星的需求。如果堅持用武僧轉，這個優先測。

### Acolyte of Chayula

不作為第一推薦。它更偏 chaos / darkness / 資源機制，可能能玩出東西，但不是召死彗星最乾淨的殼子。

### Martial Artist

目前 Martial Artist 最不適合繼續硬撐。這個角色已經明顯是攻擊身份：Killing Palm、quarterstaff、暴擊、近戰路徑。把 Martial Artist 留著去玩召死彗星，會浪費太多昇華價值。

## 最適合目標路線的職業

如果問「這條影片路線最適合誰」，答案大概率不是 Monk。

截圖路線更像：

```text
Witch / Blood Mage
```

Blood Mage 天然更接近生命、法術成本、殘留、生存資源、召喚/觸發這些主題。Monk 能改，但屬於改裝車。

## 哪些必須換

### 武器

目前 `Sinister Quarterstaff` 是攻擊武器，不屬於目標體系。

根據參考影片路線，這裡不應該說成「買 Spirit 武器」，也不應該預設換成 sceptre。更準確的目標武器身份是：

```text
主手：法杖
副手：盾牌
```

法杖負責法術輸出側：

```text
法術傷害
法術等級 / 冰法術等級，如果有
施法速度
如果最終走暴擊，找法術暴擊相關
如果走大法師，要魔力
```

盾牌負責生存側：

```text
生命
抗性
格擋 / 防禦價值
能量護盾或護甲/能盾底子
如果市場和底子允許，可以順帶帶 Spirit
```

Spirit 仍然是必須解決的預算，但它應該作為整套 build 的資源方案來解決：天賦、項鍊、胸甲、盾牌和其他裝備一起湊。不能預設「武器位必須是 Spirit 武器」。

### 項鍊

目前項鍊有：

```text
+1 to Level of all Melee Skills
```

這對召死彗星基本不對路。要換成：

```text
+Spirit
+法術 / 召喚相關等級或傷害
+生命
+抗性
+缺屬性就補屬性
```

### 空裝備槽

目前 PoB 裡這些位置是空的：

```text
胸甲
手套
鞋子
```

這是轉換前的大問題。這些槽位必須用來補生命、抗性、Spirit/資源、防禦。

### 抗性

目前抗性還不能轉：

```text
火抗：27%
冰抗：4%
電抗：-18%
混沌抗：8%
```

轉換前至少要做到：

```text
三元素抗接近 75%
優先修電抗
混沌抗至少非負
```

## 轉換前準備清單

花金幣前先準備：

```text
1. 確認 Monk 昇華能不能順利換。
2. 準備至少 300k-400k gold。
3. 準備目標技能寶石。
4. 準備目標輔助寶石。
5. 做出 120 Spirit 方案，但不要預設武器必須是 Spirit 武器。
6. 補齊胸甲、手套、鞋子。
7. 先修電抗。
8. 在 PoB 裡模擬新樹，確認 Spirit 不是負數。
9. 認真判斷是否不如重練 Witch / Blood Mage。
```

## 實戰建議

如果只是想保留武僧身份，體驗影片機制：

```text
能換就往 Invoker 轉。
不要繼續用 Martial Artist 當最終殼子。
準備 400k+ gold。
先買法杖、防禦盾牌、Spirit 方案組件和抗性裝備。
技能包湊齊後再洗樹。
```

如果想要最乾淨、最接近影片的版本：

```text
重練 Witch / Blood Mage。
```

## 最終結論

能轉，但貴，而且結構彆扭。

人話版本：目前 Martial Artist 是近戰攻擊車，目標召死彗星是法術觸發引擎。你可以把引擎焊進去，但如果目標是按影片舒服跑，Blood Mage 車架大概率更合適。
